## Why

The GUI app is currently desktop-only (Linux, Windows, macOS). Adding Android and iOS support via Tauri Mobile would let users track their sticker collection on the go, directly from their phone.

## What Changes

1. **Mobile project scaffolding**: Init Android (Gradle) and iOS (Xcode) projects via Tauri CLI
2. **Rust backend adaptation**: Replace `dirs::download_dir()` with mobile-safe app data paths; disable platform-specific commands (open folder) on mobile
3. **Tauri config updates**: Add mobile bundle targets, remove desktop-only constraints (min-width), add mobile-specific settings
4. **Capabilities**: Update or add mobile capability with mobile-compatible permissions
5. **UI responsive layout**: Convert fixed sidebar to bottom tab bar on mobile, ensure all screens are touch-friendly
6. **Icons & assets**: Add Android mipmap and iOS xcassets icons
7. **Build scripts**: Add `tauri android` and `tauri ios` scripts to package.json

## Capabilities

### New Capabilities
- `gui-android-build`: Android APK build pipeline via Tauri Mobile
- `gui-ios-build`: iOS IPA build pipeline via Tauri Mobile
- `gui-mobile-backend`: Rust backend adapted for mobile (paths, commands, capabilities)
- `gui-mobile-ui`: Responsive UI with bottom tab navigation for mobile screens

### Modified Capabilities
- `gui-export-config`: Update export flow to hide "Abrir Carpeta" on mobile and save files to app data dir

## Impact

- `src/gui/src-tauri/` — new android/ and ios/ directories (auto-generated by Tauri CLI)
- `src/gui/src-tauri/tauri.conf.json` — add mobile bundle targets, android/ios config stanzas
- `src/gui/src-tauri/capabilities/default.json` — use mobile-compatible schema
- `src/gui/src-tauri/src/lib.rs` — replace `dirs::download_dir()`, guard `open_downloads_folder` behind `#[cfg(not(target_os = "android"))]`
- `src/gui/src/components/Sidebar.tsx` — refactor to responsive sidebar/tab bar
- `src/gui/src/AppLayout.tsx` — responsive layout switch
- `src/gui/src/screens/Export.tsx` — hide "Abrir Carpeta" on mobile
- `src/gui/package.json` — add tauri android/ios scripts
- `src/gui/src-tauri/icons/` — add Android and iOS icon formats
